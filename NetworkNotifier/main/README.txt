-Προστέθηκε Λειτουργικότητα "Διατηρησης Στοιχείων Χρήστη"
	οταν γίνει login εκκινήται νεα DemoActivity που δείχνει τα στοιχεία 
	που εισήχθησαν και παρέχει κουμπι logout. Οταν όμως η εφαρμογή 
	καταστραφεί και επαννεκινηθεί, η MainActivity ανακατευθύνει αμεσα 
	στην Demo ανιχνέυοντας την υπαρξη των στοιχείων ΧΡΗΣΙΜΟΠΟΙΩΝΤΑΣ 
	SHARED PREFERENCES (εξωτερικό κρυφό αρχείο). Μονο αν γίνει logout
	επανεκκίνηση της εφαρμογής προβάλλει τη Main για να γίνει login
	
	Απαραίτητα Αρχεία: 
		 + SessionManager.java
		 βλεπε σχόλια κάθε μεθόδου για ενσωμάτωση στον MM
		
-Προστέθηκε Λειτουργικότητα "Παρακολούθησης δικτύου για αποστολή Βασης"

	ειδοποιήσεις
